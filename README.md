# Finder project
Simple CRUD for contacts

# Start
  Preferred environment `node.js@7.x` and `npm@4.x`
  ```
  npm install
  ```

# Development
  If you want to run this on LAN host, please set up data in config(.dotenv) `LAN` to your workstation IPv4(`xxx.xxx.xxx.xxx`).
  ```
  npm run dev
  ```

# Production
  ```
  npm run production
  ```
